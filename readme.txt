Консольное приложение "Телефонный справочник".

1) Запуск:
    python3 phone_directory.py

2) Для корректной работы в папке с файлом должен быть файл data.json формата:
{
    "1": {
        "Номер строки": "1",
        "Фамилия": "Иванов",
        "Имя": "Иван",
        "Отчество": "Петрович",
        "Компания": "Рога и Копыта",
        "Рабочий телефон": "001",
        "Мобильный телефон": "916-001"
    }
}
ВАЖНО: Ключами выступают номера строк

3) После запуска следуйте инструкциям в консоли
